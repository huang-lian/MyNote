- thinking  
还在思考，主观性很强，甚至是偏离事实的

- thought  
基本稳定的想法，有认真的去了解过相当一部分人的看法后形成的

- idea
关于一些「小工具」的构思

- note
多是学习笔记。编程|数学|工具用法.

- excerpt
观点、言论摘录。多是不完整的，也附加一些自己的思考

- question
有想了一点，但感觉还差一些东西。

- origin
