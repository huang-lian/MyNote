# 1.变量

1. 变量就是以一组文字或符号等，来取代一些配置或者是一串保留的数据

   显示变量内容使用 `echo ${val}`

   引用变量使用 `$`

2. 变量配置守则

   - 变量赋值使用`=`连接，中间不要空格
   - 取消变量使用`unset`

3. 配置变量时：单引号与双引号的最大不同在于双引号仍然可以保有变量的内容，但单引号内仅能是一般字符 ，而不会有特殊符号。

4. 在命令下达的过程中，反单引号(\` )这个符号代表的意义为何答在一串命令中，在 \` 之内的命令将会被先运行，而其运行出来的结果将做为外部的输入信息！
