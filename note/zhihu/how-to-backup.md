# 不同的备份方式的优缺点
## 移动硬盘、优盘
- 优点
 - 易携带 
 - 可持续读写
- 缺点
 - 成本较高
 - 保存周期较短
 - 抗干扰能力较弱

## 磁带、光盘
- 优点
 - 保存周期较移动硬盘更长
 - 成本更低
- 缺点
 - 不可再写入
 - 不便携

## 云备份
- 优点
 - 跨设备
 - 不容易丢失
- 缺点
 - 隐私性下降
 - 容易受到运营商限制，比如说下载上传限制

# 推荐的备份方式
物理+云
- 周期性备份
- 多个备份

eg
 - 可读写的作为随身备份
 - +1周期备份的硬盘或者云盘
 - +2周期的刻录备份
